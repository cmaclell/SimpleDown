SimpleDown by alexblackie

A flat-file, markdown-powered, super-simple blog engine.

SimpleDown is the simplest blog engine you will find. All you have to do is upload some Markdown files and you're already publishing. No databases, no administration, just writing...

Dependencies

PHP and a web server that can rewrite URLs (like Apache or NGINX).
